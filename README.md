# ADS - PUCPR | Exercícios e Estudos

Este repositório contém exercícios, atividades e práticas desenvolvidas durante o curso de **Análise e Desenvolvimento de Sistemas (ADS)** da **PUCPR**.

O objetivo deste repositório é registrar minha evolução ao longo do curso, armazenando códigos, experimentos e resoluções de problemas propostos nas disciplinas.

## 📚 Conteúdo

Os exercícios estão organizados por disciplina ou tema, podendo incluir:

* Raciocínio Computacional
* Lógica de Programação
* Estruturas de Dados
* Desenvolvimento Web
* Banco de Dados
* Outros conteúdos abordados durante o curso

Cada pasta contém exercícios práticos, códigos de estudo e resoluções de problemas propostos em aula.

## 🎯 Objetivo

* Praticar conceitos aprendidos no curso
* Desenvolver raciocínio lógico e computacional
* Manter um histórico de aprendizado
* Compartilhar conhecimento e evolução no GitHub

## 🛠 Tecnologias Utilizadas

Dependendo da disciplina, os exercícios podem utilizar:

* Python
* HTML
* CSS
* JavaScript
* SQL
* Outras tecnologias abordadas durante o curso

## 📌 Observação

Este repositório é voltado para fins educacionais e pode conter soluções simples ou em processo de melhoria conforme o avanço no curso.
